# SimpleTodoApp-EasyDo
 Easydo as an app that allow users add todos simply, made in swiftui, coredata, and userdefulats.

**TECH USED**
  - SWIFT
  - SWIFTUI
  - COREDATA
  - USERDEFUALTS
  - MVVM
 
 **FEATURES**
 
  - ADD TASKS EASILY 
  - UPDATE TASKS
  - ADD A PRIORITY COLOR
  - ADD A TASK CARD COLOR
  - CHNAGE THEME IN SETTINGS
